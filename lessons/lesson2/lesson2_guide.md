# P2. Игровой цикл и ввод

Цель урока: превратить неподвижную сцену в управляемую игровую сцену, где герой двигается от клавиатуры, не выходит за край окна, а программа сама повторяет шаги игры каждый кадр: прочитать ввод, обновить мир, нарисовать кадр и показать его.

## Что сделать на уроке

- Запусти `lessons/lesson2/samples/01_keyboard_game_loop.py`.
- Поменяй скорость, размер или цвет героя.
- Проверь движение стрелками или клавишами `WASD`.
- Сделай так, чтобы герой оставался внутри экрана.
- Посмотри, как герой блокируется стеной.
- Исправь `lessons/lesson2/puzzles/01_hero_escapes_screen.py`.
- Сохрани свою версию игры в `team/<team_name>/`.

## Какие файлы здесь есть

- `lesson2_slides.pdf` и `lesson2_slides.marp.md` — короткое визуальное объяснение для урока.
- `lesson2_guide.pdf` и `lesson2_guide.md` — эта памятка с нужными функциями и выражениями.
- `samples/01_keyboard_game_loop.py` — главный рабочий пример: игровой цикл, клавиатура, движение, границы и стена.
- `puzzles/01_hero_escapes_screen.py` — маленькая поломка: герой уходит за экран, нужно найти и исправить причину.
- `samples/` и `puzzles/` можно читать, запускать и менять для тренировки, но свою работу сохраняй в `team/<team_name>/`.

## Условия и свои функции

| Функции и выражения | Пример | Описание | Применил |
| --- | --- | --- | --- |
| `if` | `if game.is_key_down("right"):` | Выполняет команды, если условие верно. | ☐ |
| `elif` | `elif game.is_key_down("left"):` | Следующая проверка, если прошлое `if` не сработало. | ☐ |
| `else` | `else: speed = 0` | Что делать, если условия выше не сработали. | ☐ |
| `and` | `x > 0 and x < 800` | Оба условия должны быть верны. | ☐ |
| `or` | `key == "left" or key == "a"` | Достаточно одного верного условия. | ☐ |
| `not` | `not game.is_close_clicked()` | Переворачивает условие: “не закрыто”. | ☐ |
| `def` | `def update_world(world):` | Создаёт свою функцию, чтобы код был короче и понятнее. | ☐ |
| `return` | `return direction` | Возвращает результат из функции. | ☐ |
| `for` | `for wall in walls:` | Повторяет действие для каждого объекта в списке. | ☐ |

## Проверка границ

| Функции и выражения | Пример | Описание | Применил |
| --- | --- | --- | --- |
| `min()` | `min(x, 800)` | Не даёт числу стать больше верхней границы. | ☐ |
| `max()` | `max(x, 0)` | Не даёт числу стать меньше нижней границы. | ☐ |
| `max()` + `min()` | `x = max(0, min(x, 800))` | Зажимает число внутри диапазона. | ☐ |
| Проверка прямоугольников | `hero.x < wall.x + wall.width` | Помогает понять, касается ли герой стены. | ☐ |
