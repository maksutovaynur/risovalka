# 12. Вспомогательные игровые инструменты

Кроме движка и ассетов курсу нужны небольшие инструменты, которые помогают детям создавать свои игры.

## 1. Мини bitmap-рисовалка

Цель: рисовать простые PNG-ассеты без лишнего интерфейсного шума.

Минимальные функции:

- холст фиксированного размера;
- координатная сетка;
- отображение координат курсора;
- палитра с названиями цветов;
- рядом с цветами RGB и HEX;
- прозрачный фон;
- ластик;
- заливка;
- экспорт PNG;
- сохранение сразу в `assets/`;
- подсказки, связанные с логикой занятий.

## 2. Генератор/редактор спрайтов

Цель: удобно создавать и сравнивать кадры анимации.

Минимальные функции:

- список кадров;
- копировать предыдущий кадр;
- onion-skin/полупрозрачный предыдущий кадр (*);
- предпросмотр анимации;
- настройка скорости;
- экспорт кадров;
- экспорт описания спрайта;
- подсказка: «спрайт — это серия картинок».

## 3. Мини векторная рисовалка

Цель: создавать геометрию и получать команды для рисования.

Минимальные функции:

- координатная сетка;
- точки, линии, прямоугольники, круги, многоугольники;
- числовое редактирование координат;
- anchor/центр;
- цвет, RGB, HEX;
- экспорт SVG (*);
- генерация команды `game.draw_*` для полученной фигуры;
- копирование команды в буфер.

## 4. Инструмент achievements/leaderboard

Цель: сделать прогресс интерактивным.

Минимальные функции:

- команды/пары;
- дерево навыков;
- отметка achievements;
- leaderboard для проектора;
- badges;
- секретные achievements;
- экспорт итогов.

## 5. Приоритет реализации

Сначала:

1. leaderboard/achievements;
2. bitmap-рисовалка;
3. спрайт-редактор;
4. векторная рисовалка.
