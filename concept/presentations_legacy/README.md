# Legacy presentations

Эта папка содержит ранние черновики презентаций `P0`, `P1`, `P2`.

Они оставлены как концептуальный архив и источник отдельных идей, но не являются актуальными материалами уроков. Реальные материалы первого урока находятся в:

```text
lessons/lesson1/
```

Новые материалы уроков нужно создавать и обновлять в `lessons/`, а не в этой legacy-папке.
